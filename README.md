# TheCafeProject

for preview:
[Demo](https://rkymtc.github.io/TheCafeProject/)

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/TheCafeProject.git
```

Proje dizinine gidin

```bash
  cd TheCafeProject
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```
